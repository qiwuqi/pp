# Proxy Fetcher

🎯 一个简单的 Node.js 脚本，用于从 proxy.scdn.io 获取 HTTPS 代理列表，并随机保存 10 个代理到本地文件 `proxy-list.txt`。

## ✨ 功能亮点

- ✅ 自动抓取 20 个代理 IP
- 🎲 随机选取其中 10 个代理
- 💾 输出结果保存到本地 `proxy-list.txt`
- 📦 Node.js 原生支持，使用现代 `ESM` 模块
